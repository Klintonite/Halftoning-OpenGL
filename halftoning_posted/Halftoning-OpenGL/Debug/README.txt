Halftoning
by: Klinton Shmeit

This assignment performs halftoning on an image in 3 different ways. Dither, Floyd and Steinberg, and Dot Diffuse.
To use the dither, type in the image name and then choose "1";
To use the Floyd and Steinberg, type in the image name and then choose "2" and choose the threshhold value from 1-255;
To use the dither, type in the image name and then choose "3" and choose the threshhold value from 1-255 then choose the matrix you want to use by pressing 1 or 2;
